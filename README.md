## Getting Started

First, run the development server:

```bash

```

## DOCKER
```bash
docker-compose down -v
docker-compose build --no-cache
docker-compose up
#or only file run
docker-compose -f docker-compose-dev.yml down -v
docker-compose -f docker-compose-dev.yml up --build
```
